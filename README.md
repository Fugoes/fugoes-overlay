# fugoes-overlay

This is Fugoes's personal gentoo Linux overlay.

## How to use

```bash
layman -o https://raw.githubusercontent.com/Fugoes/fugoes-overlay/master/overlay.xml -f -a fugoes
```

www-apps/grafana-bin
